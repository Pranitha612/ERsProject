Dataset Folder Note
===================

This folder is intended for sample test/input images only.

This project does not use a custom training dataset because it uses pretrained
DeepFace and OpenCV models.

Recommended contents for submission:

datasets/
  sample_test_images/
    test.jpg
    group_sample_1.jpg

Public reference dataset links are listed in the main README.md file:

- FER2013 for emotion recognition reference
- UTKFace for age/gender reference
- Adience for age/gender benchmark reference
- OpenCV Haar cascades for pretrained XML cascade files

Do not claim that the project was trained on these datasets unless a separate
training process was actually performed.
