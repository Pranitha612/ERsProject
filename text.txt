**Important Clarifications**
- **Face ID modelling:** there is **no face recognition / face identification model** in this project.
  The `person_id` is just assigned as `i + 1` for each detected face in the current image in [backend/model.py](/abs/path/c:/Users/Ritwik/ERsProject/backend/model.py:283).  
  So this is **not** FaceNet, ArcFace, VGG-Face, or any identity-matching model.
- **Pair-wise relationship classification:** there is **no trained ML model** here.
  It is a **rule-based classifier** implemented in `predict_relationship()` in [backend/model.py](/abs/path/c:/Users/Ritwik/ERsProject/backend/model.py:216), using roles like `child`, `adult`, and `elder`.
- **Graph generation:** there is **no graph neural network or graph ML model**.
